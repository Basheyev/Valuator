Шаг 1. Оценка ожидаемой exit-стоимости компании (Exit Value)

Оценка ожидаемой exit-стоимости компании осуществляется на основе трёх основных методов на год выхода:

№ Метод Описание метода оценки стоимости Формула расчета
1. EBITDA multiple Метод кратного EBITDA - самый быстрый способ оценки exit-стоимости на этапе soft-commitment. Используется EBITDA последнего года перед выходом и мультипликатор (M) от 4 до 8. Exit Value = EBITDA × M
Где M=4: для успешных компаний, M=6 применяются для быстрорастущих компаний или M=8 для лидеров рынка с высокой прибыльностью.
2. Multiples Метод мультипликаторов - использование средних мультипликаторов (K) ряда аналогичных публичных компаний для оценки прогнозной стоимости:
- EV / EBITDA (Enterprise Value to EBITDA): Стоимость предприятия к EBITDA.
- EV / Sales (Enterprise Value to Sales): Стоимость предприятия к выручке. 		EV value = EBITDA Kebitda
EV Value = SALES × Ksales
3. Discounted Cash Flow (DCF) Дисконтирование денежного потока. Оцениваем на основании заявленных показателей свободного денежного потока на прогнозируемый период. Средневзвешенная стоимость капитала:

WACC = (Equity / (Equity + Debt)) × re + (Debt / (Equity + Debt)) × rd × (1 - Tc)

где, Debt - совокупная сумма долговых обязательств Equity - совокупная сумма инвестиций в капитал Re - ставка по инвестициям в капитал Rd - ставка по долговым обязательствам Tc - налоговая ставка (~20% КПН Казахстан) 		Exit Value = Σ (FCFt / (1 + WACC)t) + TV - NFP
TV = (FCFn × (1 + g)) / (WACC - g)

где:
NFP = Долги - Деньги и их эквиваленты
TV = Терминальная стоимость
n - количество прогнозируемых лет
g - средневзвешенный рост экономики (~5%)

Итоговый Exit Value - это средняя целевая оценка по перечисленным методам оценки

Шаг 2. Дисконтирование exit-стоимости до сегодняшней оценки (VC)

Описание метода Формулы расчета
Текущая стоимость до инвестирования денег (pre-money Present Value) оценивается путём дисконтирования от целевой exit стоимости (Exit Value) с учётом 4-6х роста за 2-3 года (r=58%-82% годовых, n = 2, 3). Present Value = Exit Value / (1 + r)n

r - требуемая доходность (58% = 4x за 3 года, 82% = 6x за 3 года), n - число лет до выхода.
